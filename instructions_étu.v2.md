
## Pré-requis

- avoir installé Pluto (voir sous Moodle la section **En Amont du Projet**)

## Première ouverture du notebook

1. dés-archivez l'archive code_projet_cs.tar.gz
2. allez dans le répertoire code_projet_cs
3. lancez julia
4. lancez Pluto (vous avez déjà dû le faire lors de la préparation)
5. dans votre navigateur, dans l'onglet Pluto, ouvrir le notebook _Pluto_file.jl_
6. Exécutez le code du notebook (en haut à droite) ; vous pouvez suivre l'avancement en cliquant sur l'icône _statut_ ; La première exécution (et seulement celle-là) prend pas mal de temps (plus de 10 mn). Vous pouvez commencer à lire les explications (étape suivante).

## Vous pouvez maintenant lire les explications et répondre aux questions directement dans le notebook (cellules dédiées et cellules codes)

## N'oubliez pas de sauvegarder (Ctrl+s) 

- toutes les modifications seront inclues dans le fichier _Pluto_file.jl_
- c'est ce fichier que vous déposerez sous Moodle
